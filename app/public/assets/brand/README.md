# Invoice Baba - Brand Assets

Place the following logo files in this directory:

| File Name | Description |
|-----------|-------------|
| `icon.png` | App icon (document with checkmark) - white background, square |
| `icon-transparent.png` | App icon (document with checkmark) - transparent background, square |
| `logo-full.png` | Full logo with "InvoiceBaba - Billing Made Easy" text - white background |
| `logo-full-transparent.png` | Full logo with "InvoiceBaba - Billing Made Easy" text - transparent background |
| `favicon.png` | Favicon for browser tab (use the icon, ideally 32x32 or 64x64) |
